# RESULTADOS OBTENIDOS
- Durante el desarrollo de esta práctica se logró comprender el funcionamiento de los algoritmos de ordenamiento por Inserción y Shell, aplicándolos sobre un arreglo de enteros predefinido. Se pudo observar cómo cada algoritmo organiza los datos paso a paso, mostrando las comparaciones, cambios e iteraciones realizadas en cada proceso. Esto permitió analizar el comportamiento interno de cada método y cómo van transformando el arreglo hasta obtener el resultado ordenado. Además, se identificaron diferencias importantes entre ambos algoritmos. El método de Inserción trabaja de forma más directa comparando elementos consecutivos, mientras que Shell optimiza el proceso utilizando saltos (gap), lo que reduce el número de movimientos en ciertos casos.
---
# CONCLUSIONES 
- Se logró comprender el procedimiento de los algoritmos de ordenamiento, especialmente cómo funcionan paso a paso mediante comparaciones y desplazamientos de elementos.
- Se identificó que el algoritmo de Shell puede ser más eficiente que Inserción en ciertos casos, debido a que reduce la cantidad de movimientos utilizando saltos, aunque ambos llegan al mismo resultado final.
- El análisis de comparaciones, cambios e iteraciones permitió evaluar el rendimiento de cada algoritmo y entender cuál puede ser más conveniente dependiendo del tamaño y estado del arreglo.
